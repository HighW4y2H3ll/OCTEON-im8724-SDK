README for Cavium Inc OCTEON SDK docs directory

This directory contains documentation for the OCTEON SDK.


Contents:
---------------------------------------------------------------------
README.txt	this file
html/		HTML documentation for the SDK.  Most of this is generated
		using Doxygen.  This documentation covers the simulator,
		OCTEON simple executive, debugger, and bootloader.
		The file html/index.html is the main page, and should be used
		as the starting point for browsing the directory.

The following directories contain the original documentation and licenses for
the open source components of the SDK.
---------------------------------------------------------------------
		
u-boot		documentation for the u-boot bootloader.
newlib		documentation for the newlib C library.
gcc		documentation for the gcc toolchain.
ddd		documentation for the ddd debugger.
